Lachlan Neilsen: Project

Pathfinding within hgazardous areas:

My project idea is to build a system for planning routes through hazardous areas, for robots and drones. This would utilize a neural network to analyse drone footage and gyroscopic data, recording the terrain as a 3D model, which is used to plot a safe path for more expensive drones to follow.

 

A hazardous area is a physical space in which explosive atmospheres may occur. Click here to read more about hazardous areas, and their definitions.
